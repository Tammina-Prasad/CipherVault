About CipherVault

Cryptography is the practice of securing information so that only intended parties can read it. It powers everything from messaging apps to online banking.

Data security matters because every leaked password or message can lead to identity theft, financial loss, and broken trust.

CipherVault is a lightweight demonstration of symmetric encryption using Python's cryptography library (Fernet / AES-128-CBC + HMAC). Type, encrypt, share, decrypt — safely.
